
# HarryPotter.Books.Api

This Api return harry potter books.

## About API

#### Get all books

```http
  GET /api/v1
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `CorrelationId` | `Guid` | Tracking  |
